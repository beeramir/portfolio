Twitch API

This app utilizes the API for the popular game streaming platform Twitch. It loops through an array of streamers displaying their current online/offline status, what game they are playing, as well as their current live viewer count.

Challenges:
This was my first foray into asynchronous data, so my program was trying to display data before it was actually received. I solved this by learning more about deferred callbacks.
